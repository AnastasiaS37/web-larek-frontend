# Web-Larek – Online Shop

Web-Larek is a web application for an online mock store with study-related products.

This project focuses on building a modular frontend architecture using TypeScript and a component-based MVP approach.

## 📌 Key Features

* Structured architecture based on MVP pattern
* Shopping cart functionality
* Order checkout with form validation
* Modal windows for products, cart, and checkout flow

## 🛠️ Tech Stack

* TypeScript / JavaScript
* HTML5 / CSS3

## 👩‍💻 My Contributions

* Implemented order form validation
* Organized project using MVP architecture
